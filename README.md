# Personal Portfolio - Capstone Project 2

This is my second Capstone Project for the Full Stack Web Development Bootcamp. The goal of this project was to build a professional personal website from scratch to showcase my skills, background, and projects as a developer.

## 🚀 Live Demo

[tomi-cancheff.github.io/personal-site](https://tomi-cancheff.github.io/personal-site/)

## 🛠️ Built With

- **HTML5** — Semantic markup for structure
- **CSS3** — Custom external stylesheet for advanced styling, including radial gradients and custom color palettes
- **Bootstrap 5** — Utilized for the responsive grid system, cards, and pre-built components like buttons and layouts

## ✨ Key Features

- **Hero Section** — Features a custom radial gradient background and a profile image with optimized spacing for high visual impact
- **About Me** — A clean section highlighting core skills (Web Development, Game Design) using a high-contrast color scheme (`#111844` background and `#EAE0CF` text)
- **Project Gallery** — A responsive grid of 3-column cards using the Bootstrap Grid System. Each card includes:
  - Project descriptions
  - "Call to Action" (CTA) buttons aligned at the bottom using Flexbox (`mt-auto`)
- **Online Resume** — A dedicated section with a clear link to my external web resume
- **Contact Section** — A functional contact area with a `mailto` button that automatically opens the user's email client to reach me directly at [tomicancheff@gmail.com](mailto:tomicancheff@gmail.com)

## 🎨 Design Principles Applied

During development, I focused on several core UI/UX principles:

- **Hierarchy** — Used typography (Montserrat) and color contrast to guide the user's attention to the most important information first
- **Consistency** — Maintained a uniform color palette and button styles throughout the site to ensure a smooth user experience
- **Responsiveness** — The site is fully mobile-responsive, adapting its layout for desktops, tablets, and smartphones using Bootstrap breakpoints
- **White Space** — Injected deliberate spacing (padding and margins) to create a clean and professional feel
